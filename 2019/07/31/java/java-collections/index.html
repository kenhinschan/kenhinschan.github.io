List HashMap Set зәҝзЁӢе®үе…ЁеЈ°жҳҺдҪҝз”Ёж–№жі•

  • 1.дҪҝз”ЁCollections. жһ„е»әеҗҢжӯҘ
  • 2гҖҒдҪҝз”ЁThreadLocalе°ҒиЈ…

HashMapеӯҳеӮЁеҺҹзҗҶ

  • HashMapеӯҳеӮЁз»“жһ„жҳҜпјҡж•°з»„пјҲbucketпјү + еҚ•еҗ‘й“ҫиЎЁ пјҲжәҗз Ғпјҡtransient Node<K,V>[] table;пјү

  • map.put(key,value) еӯҳеӮЁй”®еҖјеҜ№

  • дёәд»Җд№ҲString, Intergerиҝҷж ·зҡ„wrapperзұ»йҖӮеҗҲдҪңдёәй”®пјҹеӣ дёәStringжҳҜдёҚеҸҜеҸҳзҡ„пјҢд№ҹжҳҜfinalзҡ„пјҢиҖҢдё”е·Із»ҸйҮҚеҶҷдәҶequals()е’ҢhashCode()ж–№жі•дәҶпјҢеҮҸе°‘hashзў°ж’һ

  • ж №жҚ®keyзҡ„hashcodeеҖји®Ўз®—пјҲи®Ўз®—жі•еҲҷжҳҜпјҡдҪҺ16дҪҚгҖҒй«ҳ16дҪҚ з„¶еҗҺејӮжҲ–иҝҗз®—пјүеҮәж•°з»„дёӢж Ү

  • жҠҠkey,valueеӯҳеӮЁеҲ°й“ҫиЎЁеҪ“дёӯ

  • ж №жҚ®keyиҺ·еҸ–еҖјеҫ—еҺҹзҗҶпјҡ ж №жҚ®keyзҡ„hashcodeи®Ўз®—еҮәж•°з»„дёӢж ҮпјҢз„¶еҗҺйҒҚеҺҶйҒҚеҺҶй“ҫиЎЁ

  • hashcodeзӣёеҗҢзҡ„keyпјҲhashзў°ж’һпјүе°ұиҰҒйҒҚеҺҶй“ҫиЎЁпјҢеҲ©з”Ёkey.equals() жқҘжүҫеҲ°еӯҳеӮЁзҡ„еҜ№иұЎ

  • HashMapдҪҝз”Ёй“ҫиЎЁжқҘи§ЈеҶізў°ж’һй—®йўҳпјҢеҪ“еҸ‘з”ҹзў°ж’һдәҶпјҢеҜ№иұЎе°ҶдјҡеӮЁеӯҳеңЁй“ҫиЎЁзҡ„дёӢдёҖдёӘиҠӮзӮ№дёӯгҖӮ HashMapеңЁжҜҸдёӘй“ҫиЎЁиҠӮзӮ№дёӯеӮЁеӯҳй”®еҖјеҜ№еҜ№иұЎ

  • еҪ“дёӨдёӘдёҚеҗҢзҡ„й”®еҜ№иұЎзҡ„hashcodeзӣёеҗҢж—¶дјҡеҸ‘з”ҹд»Җд№Ҳпјҹ е®ғд»¬дјҡеӮЁеӯҳеңЁеҗҢдёҖдёӘbucketдҪҚзҪ®зҡ„й“ҫиЎЁдёӯгҖӮй”®еҜ№иұЎзҡ„equals()ж–№жі•з”ЁжқҘжүҫеҲ°й”®еҖјеҜ№гҖӮ

  • й“ҫиЎЁй•ҝеәҰеӨ§дәҺ8пјҢдјҡе°ҶеҚ•еҗ‘й“ҫиЎЁз»“жһ„вҖ”-иҪ¬дёәвҖ”зәўй»‘ж ‘з»“жһ„пјҲзәўй»‘ж ‘пјҡTreeNode<K,V> е·ҰеҸіеҖјпјү

  • й“ҫиЎЁй•ҝеәҰе°ҸдәҺ6пјҢдјҡе°Ҷзәўй»‘ж ‘з»“жһ„вҖ”-иҪ¬дёәвҖ”еҚ•еҗ‘й“ҫиЎЁз»“жһ„пјҲеҚ•еҗ‘й“ҫиЎЁпјҡNode<K,V> еҸҜд»ҘжҹҘзңӢжәҗз Ғпјү

  • put()ж–№жі•зҡ„жү§иЎҢиҝҮзЁӢеҰӮдёӢжүҖзӨәпјҡ

HashMapжү©е®№

  • еҰӮжһңHashMapзҡ„еӨ§е°Ҹи¶…иҝҮдәҶиҙҹиҪҪеӣ еӯҗ(load factor)е®ҡд№үзҡ„е®№йҮҸпјҢжҖҺд№ҲеҠһпјҹ

    й»ҳи®Өзҡ„иҙҹиҪҪеӣ еӯҗеӨ§е°Ҹдёә0.75пјҢд№ҹе°ұжҳҜиҜҙпјҢеҪ“дёҖдёӘmapеЎ«ж»ЎдәҶ75%зҡ„bucketж—¶еҖҷпјҢ
е’Ңе…¶е®ғйӣҶеҗҲзұ»(еҰӮArrayListзӯү)дёҖж ·пјҢе°ҶдјҡеҲӣе»әеҺҹжқҘHashMapеӨ§е°Ҹзҡ„ 2еҖҚзҡ„bucketж•°з»„пјҢ
жқҘйҮҚж–°и°ғж•ҙmapзҡ„еӨ§е°ҸпјҢе№¶е°ҶеҺҹжқҘзҡ„еҜ№иұЎж”ҫе…Ҙж–°зҡ„bucketж•°з»„дёӯгҖӮиҝҷдёӘиҝҮзЁӢеҸ«дҪңrehashingпјҢ
еӣ дёәе®ғи°ғз”Ёhashж–№жі•жүҫеҲ°ж–°зҡ„bucketдҪҚзҪ®

  • пјҲjdk1.7дјҡжңүиҝҷз§Қжғ…еҶөпјҢjdk1.8е·Із»ҸеҒҡдәҶдјҳеҢ–пјүеӨҡзәҝзЁӢдёӢпјҢдёӨдёӘзәҝзЁӢйғҪеҸ‘зҺ°HashMapйңҖиҰҒйҮҚж–°и°ғж•ҙеӨ§е°ҸдәҶпјҢдјҡеҸ‘з”ҹжқЎд»¶з«һдәүпјҢйҖ жҲҗжӯ»еҫӘзҺҜ

    еҪ“йҮҚж–°и°ғж•ҙHashMapеӨ§е°Ҹзҡ„ж—¶еҖҷпјҢзЎ®е®һеӯҳеңЁжқЎд»¶з«һдәүпјҢеӣ дёәеҰӮжһңдёӨдёӘзәҝзЁӢйғҪеҸ‘зҺ°HashMapйңҖиҰҒйҮҚж–°и°ғж•ҙеӨ§е°ҸдәҶпјҢ
е®ғд»¬дјҡеҗҢж—¶иҜ•зқҖи°ғж•ҙеӨ§е°ҸгҖӮеңЁи°ғж•ҙеӨ§е°Ҹзҡ„иҝҮзЁӢдёӯпјҢеӯҳеӮЁеңЁй“ҫиЎЁдёӯзҡ„е…ғзҙ зҡ„ж¬ЎеәҸдјҡеҸҚиҝҮжқҘпјҢ
еӣ дёәз§»еҠЁеҲ°ж–°зҡ„bucketдҪҚзҪ®зҡ„ж—¶еҖҷпјҢHashMapе№¶дёҚдјҡе°Ҷе…ғзҙ ж”ҫеңЁй“ҫиЎЁзҡ„е°ҫйғЁпјҢиҖҢжҳҜж”ҫеңЁеӨҙйғЁпјҢ
иҝҷжҳҜдёәдәҶйҒҝе…Қе°ҫйғЁйҒҚеҺҶ(tail traversing)гҖӮеҰӮжһңжқЎд»¶з«һдәүеҸ‘з”ҹдәҶпјҢйӮЈд№Ҳе°ұжӯ»еҫӘзҺҜдәҶ

  • еӨҡзәҝзЁӢдёӢдҪҝз”Ё ConcurrentHashMapпјҲеј•е…ҘдәҶеҲҶж®өй”Ғпјү
