5、常用docker命令

  • 1、修改镜像文件拉取地址为ustc （也可以登录阿里云获取私有的镜像加速地址）

    • vi /etc/docker/daemon.json

      • 配置生效 - 命令：sudo systemctl daemon-reload

  • 2、重启docker - 命令：systemctl restart docker

  • 3、停止docker - 命令：service docker stop

  • 4、列出镜像列表 - 命令： docker images

  • 5、查看所有容器 - 命令：docker ps -a

    • 查看正在运行的容器 - 命令：docker ps

    • docker ps -l

  • 6、拉取镜像 - 命令：docker pull mysql:8.0

  • 7、启动docker中镜像 - 命令：docker run –name mysql8.0 -p 3307:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql:8.0

  • 8、删除容器 - 命令：docker rm <容器名 or ID>

  • 9、查看容器日志 - 命令：docker logs -f <容器名 or ID>

  • 10、停止、启动、杀死指定容器

  • 11、后台运行 - 命令：docker run -d

  • 12、暴露端口

  • 13、docker容器与宿主机的拷贝
  • 14、查看容器挂载目录
  • 15、提交容器为新镜像

运行容器

  • 容器可以是springboot程序

  • 命令：docker run hello-world （这里的hello-world是springboot应用程序）
