Java的静态与非静态

2019-09-15

Java的静态与非静态

  • 静态成员、静态方法随着类的加载而加载（先于非静态存在jvm内存中）

  • 静态成员、静态方法可以通过类名直接调用（类名.静态方法/成员变量）

  • 静态方法不能调用非静态的方法、非静态的成员变量（编译不通过，因为静态先于非静态存在jvm内存中）

  • 非静态成员方法、成员变量必须通过对象实例调用，或者反射调用

（静态依属于类，非静态依属于类的对象）
（静态先于非静态存在jvm内存中）
