jvm性能调休工具使用

  • jvisualVM（jdk自带的）

    • 在终端运行命令：jvisualvm（就可以打开）

  • jconsole （JDK自带的）

    • 在终端运行命令：jconsole （就可以打开）

    • 本地进程（选择进程–连接–查看）、远程进程监控（需要用户、密码）

    • 远程进程监控

      • 需要在服务器上开启配置
  • 视图界面：

  • 启动： systemctl start firewalld
  • 查状态：systemctl status firewalld
  • 停止： systemctl disable firewalld
  • 禁用： systemctl stop firewalld
