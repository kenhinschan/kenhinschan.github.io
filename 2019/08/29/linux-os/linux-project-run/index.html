2019-08-29

项目线上运行突然变慢了

  • 什么原因导致项目运行变慢？

  • 项目肯定出现了问题（这点毋庸置疑）

  • 程序问题导致（日志分析，jvm内存监控，服务器内存监控）

  • sql问题导致，数量增大导致 （慢查询分析）

  • 高并发环境下造成，一台服务器扛不住压力 （考虑集群、nginx做负载均衡）

怎样排查、定位问题所在

  • 首先想到的是查看日志、看有没有异常

  • 查看服务器cpu、内存使用情况，或者监控jvm内存使用情况，服务器压力大

    • 可能是高并发造成的

    • 可能程序问题

  • 开启慢查询分析（加索引优化sql）

解决方案

  • 定位到程序问题，优化程序

  • 定位到sql问题，sql优化、索引优化

  • 定位到高并发问题，加redis缓存

  • 定位到业务急速发展，带来大访问量；准备集群，并做负载均衡处理
